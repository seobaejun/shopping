# 10쇼핑게임 프로젝트 완료 보고서

## 📋 프로젝트 개요

**프로젝트명**: 10쇼핑게임 온라인 쇼핑몰  
**개발 기간**: 2026년 2월  
**기술 스택**: HTML5, CSS3, JavaScript (Vanilla), Firebase  
**반응형 지원**: 데스크톱, 태블릿, 모바일 (완전 반응형)

---

## 🎯 구현된 주요 기능

### 1. 10쇼핑게임 추첨 시스템
- **10명 1조 추첨 시스템**: 상품별로 10명이 모이면 자동으로 1개 조 구성
- **당첨 비율**: 조당 2명 당첨 (20% 확률)
- **쇼핑 지원금**: 미선정 8명에게 자동 계산 및 지급
- **자동/수동 추첨**: 관리자가 선택 가능
- **회차 관리**: 각 조마다 회차 번호 자동 부여
- **공정한 추첨**: 암호학적 난수 생성 알고리즘 사용

## 🔧 기술적 구현 사항

### 백엔드
- **Firebase Authentication**: 사용자 인증
- **Firestore Database**: 데이터베이스
  - 회원 정보 (users)
  - 상품 정보 (products)
  - 주문 정보 (orders)
  - 추첨 정보 (lottery)
  - 게시판 (posts)
  - 알림 (notifications)
  - 카테고리 (categories)

### 프론트엔드
- **HTML5**: 시맨틱 마크업
- **CSS3**: 
  - CSS Variables (색상, 폰트 등)
  - Flexbox & Grid Layout
  - 반응형 디자인 (Media Queries)
  - 트랜지션 & 애니메이션
- **JavaScript (Vanilla)**:
  - 모듈화된 코드 구조
  - 이벤트 위임
  - 비동기 처리 (Promise, async/await)
  - 로컬스토리지 활용

### 반응형 디자인
- **데스크톱**: 1024px 이상
- **태블릿**: 768px ~ 1023px
- **모바일**: 480px ~ 767px
- **소형 모바일**: 480px 미만

### 브라우저 호환성
- Chrome, Edge, Firefox, Safari 최신 버전 지원
- 모바일 브라우저 지원 (iOS Safari, Chrome Mobile)

## 🚀 서버 실행 방법

### Python을 사용한 서버 실행 (권장)

```bash
# 1. 프로젝트 디렉토리로 이동
cd C:\10-2\shopping

# 2. Python HTTP 서버 실행 (포트 8000)
python -m http.server 8000

# 3. 브라우저에서 접속
# 메인 페이지: http://localhost:8000/index.html
# 관리자 페이지: http://localhost:8000/admin/index.html
```

### Node.js를 사용한 서버 실행 (대안)

```bash
# http-server 설치 (전역)
npm install -g http-server

# 서버 실행
cd C:\10-2\shopping
http-server -p 8000

# 브라우저에서 http://localhost:8000 접속
```
