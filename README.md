# 10쇼핑게임 - 온라인 쇼핑몰

> ⚠️ **이미지 안내**: 현재 모든 상품 이미지는 플레이스홀더(더미 이미지)를 사용하고 있습니다.  
> 실제 상품 이미지는 관리자 페이지의 **상품관리 → 상품등록/수정**에서 업로드하실 수 있습니다.

> 📦 **카테고리 안내**: 현재는 상품명으로 카테고리를 자동 추측합니다.  
> 실제 상품 업로드 시 `category` 필드를 추가하면 정확한 카테고리가 반영됩니다.
> 
> ```javascript
> {
>     title: '애견 영양 간식 치킨맛 200g',
>     category: '반려동물',  // ← 추가!
>     support: '1,500원',
>     // ...
> }
> ```

https://10-2market.com/ 사이트를 분석하여 똑같이 구현한 쇼핑몰 웹사이트입니다.

## 프로젝트 구조

```
shopping/
├── index.html          # 메인 HTML 파일
├── css/
│   └── style.css      # 스타일시트
├── js/
│   └── script.js      # JavaScript 파일
├── screenshots/       # 스크린샷 폴더
└── README.md          # 프로젝트 설명
```

## 주요 기능

### 1. 헤더 & 네비게이션
- 상단 유틸 메뉴 (주문내역, FAQ, 1:1문의, 사용후기, 상품문의)
- 로고 및 검색창
- 사용자 메뉴 (로그인, 회원가입)
- 전체 카테고리 버튼
- 메인 네비게이션 (히트상품, 추천상품, 최신상품, 인기상품)
- 인기 검색어 토글

### 2. 카테고리 사이드바
- 전체 카테고리 목록
  - 반려동물 (강아지/고양이 사료, 간식, 용품)
  - 식품
  - 건강보조식품
  - 화장품
  - 생활용품
  - 가전제품
  - 선물용품
  - e-쿠폰 (커피/음료, 치킨/피자, 베이커리)
- 고객센터 정보
- 빠른 링크 (공지사항, 자주묻는질문, 마이페이지)

### 3. 메인 슬라이더
- 자동 슬라이드 (5초 간격)
- 이전/다음 버튼
- 슬라이드 인디케이터 (도트)

### 4. 상품 섹션
각 섹션마다 상품 카드 그리드로 구성:
- **히트상품 🔥**: 인기있는 상품들
- **추천상품 👍**: 추천하는 상품들
- **최신상품 ✨**: 새로 등록된 상품들
- **인기상품 ❤️**: 많이 팔린 상품들

#### 상품 카드 구성
- 상품 이미지
- 배지 (히트/최신/추천/인기)
- 상품명
- 옵션 정보
- 쇼핑지원금
- 고객평점
- 공유하기 버튼

### 5. 사용후기 섹션
- 고객 리뷰 표시 영역
- 현재: "등록된 사용후기가 없습니다" 메시지

### 6. 비디오 섹션
- 홍보 영상 임베드

### 7. 푸터
- 회사 정보 링크
- 고객센터 정보 (전화번호, 영업시간)
- 결제 정보 (계좌번호)
- 배송 정보
- QUICK 링크
- 앱 다운로드 링크
- 회사 상세 정보
- 저작권 정보

### 8. 모바일 기능
- 하단 고정 네비게이션 (전체메뉴, 검색, 홈, 로그인, 찜)
- 반응형 디자인 (768px, 480px 브레이크포인트)

### 9. 추가 기능
- 팝업 (오늘 하루 그만보기 기능)
- 오늘 본 상품 사이드 위젯
- 스크롤 시 헤더 그림자 효과

## 기술 스택

- **HTML5**: 시맨틱 마크업
- **CSS3**: 
  - CSS Variables (색상, 폰트 등)
  - Flexbox & Grid Layout
  - 반응형 디자인 (Media Queries)
  - 트랜지션 & 애니메이션
- **JavaScript (Vanilla)**:
  - 슬라이더 자동 재생
  - 카테고리 사이드바 토글
  - 팝업 관리 (로컬스토리지)
  - 상품 동적 렌더링
  - 이벤트 위임
- **Font Awesome**: 아이콘

## 실행 방법

### 로컬 서버로 실행

1. Python HTTP 서버:
```bash
python -m http.server 8080
```

2. 브라우저에서 접속:
```
http://localhost:8080/shopping/index.html
```

### Live Server (VS Code)
1. Live Server 확장 프로그램 설치
2. `index.html` 우클릭 → "Open with Live Server"

## 반응형 브레이크포인트

- **Desktop**: 1024px 이상
- **Tablet**: 768px ~ 1023px
- **Mobile**: 480px ~ 767px
- **Small Mobile**: 480px 미만

## 구현된 상세 기능

### JavaScript 기능
1. **팝업 관리**
   - 로컬스토리지로 "오늘 하루 그만보기" 저장
   - 날짜별 팝업 표시 여부 제어

2. **슬라이더**
   - 자동 슬라이드 (5초 간격)
   - 수동 제어 (이전/다음 버튼, 도트)
   - 무한 루프

3. **카테고리 사이드바**
   - 열기/닫기 애니메이션
   - 서브메뉴 토글

4. **상품 렌더링**
   - 데이터 기반 동적 생성
   - 배지 자동 생성

5. **스크롤 효과**
   - 헤더 그림자 변화

### CSS 기능
1. **반응형 그리드**
   - 데스크톱: 4열
   - 태블릿: 3열
   - 모바일: 2열
   - 작은 모바일: 1열

2. **트랜지션**
   - 모든 인터랙티브 요소에 부드러운 전환
   - 호버 효과

3. **레이아웃**
   - Sticky 헤더 & 네비게이션
   - Fixed 하단 네비게이션 (모바일)
   - Flexbox & Grid 혼합 사용
