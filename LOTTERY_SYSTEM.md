## 🎯 시스템 개요

### 핵심 개념
- **10명 1조**: 상품별로 10명이 모이면 1개 조 구성
- **당첨 인원**: 조당 2명 당첨 (실제 구매)
- **미선정 인원**: 조당 8명 미선정 (쇼핑 지원금 지급)
- **순환 시스템**: 30명이면 10명씩 3조로 나눠서 순차 추첨

### 예시
```
상품 A 구매 신청자: 30명
  ↓
1조: 10명 → 추첨 → 당첨 2명 / 미선정 8명
  ↓
2조: 10명 (기존 20명 + 신규 2명) → 추첨 → 당첨 2명 / 미선정 8명
  ↓
3조: 10명 (기존 10명 + 신규 2명) → 추첨 → 당첨 2명 / 미선정 8명
```

## 🎲 조별 추첨 시스템

### 추첨 단위
- **상품별 추첨**: 각 상품마다 독립적으로 추첨
- **조별 추첨**: 10명이 1개 조
- **회차 관리**: 각 조마다 회차 번호 부여

### 추첨 방식

#### 자동 추첨
```javascript
조건: 추첨 대기 인원 >= 10명
실행: 자동으로 추첨 시작
결과: 즉시 당첨자 선정 및 지원금 지급
```

#### 수동 추첨
```javascript
조건: 관리자가 원하는 시점
실행: "조별 추첨하기" 버튼 클릭
결과: 현재 대기 중인 인원으로 추첨 (10명 미만도 가능)
```

### 당첨자 선정
```
전체 10명 중 랜덤으로 2명 선정
확률: 20% (2/10)
방식: 공정한 난수 생성 알고리즘 사용
```

## 💰 쇼핑 지원금 계산

### 📐 계산 공식

```
나의 쇼핑 지원금 = 

  [당첨된 2명의 제품 표기 쇼핑지원금 합계]
  ────────────────────────────────────── × [나의 구매 참여금]
  [미선정 8명의 총 구매 참여금]
```

### 📊 계산 예시

#### 예시 1: 같은 금액 상품
```
상품: 메가커피 30,000원 (쇼핑지원금 1,500원)
조 구성: 10명 모두 30,000원 구매

당첨 2명의 쇼핑지원금 합계: 1,500원 × 2 = 3,000원
미선정 8명의 총 구매금: 30,000원 × 8 = 240,000원

나의 구매금: 30,000원
나의 쇼핑 지원금 = (3,000 ÷ 240,000) × 30,000
                  = 0.0125 × 30,000
                  = 375원
                  
10원 단위 절삭: 370원 지급
```

#### 예시 2: 다른 금액 상품 (옵션 차이)
```
상품: 스타벅스 기프티콘
  - A 옵션: 10,000원 (지원금 500원) - 5명
  - B 옵션: 20,000원 (지원금 1,000원) - 5명

당첨자:
  - 1명: A 옵션 10,000원 (지원금 500원)
  - 1명: B 옵션 20,000원 (지원금 1,000원)
  당첨 지원금 합계: 1,500원

미선정 8명:
  - 4명: A 옵션 = 40,000원
  - 4명: B 옵션 = 80,000원
  총 구매금: 120,000원

나의 구매금이 A 옵션 10,000원인 경우:
나의 지원금 = (1,500 ÷ 120,000) × 10,000
            = 0.0125 × 10,000
            = 125원
            
10원 단위 절삭: 120원 지급
```

### 💡 지원금 규칙

1. **10원 단위 절삭**
   ```
   12,399원 → 12,390원 지급
   5,674원 → 5,670원 지급
   1,235원 → 1,230원 지급
   ```

2. **당일 지급**
   ```
   추첨 완료 후 당일 지급
   상품별로 합산하여 지급
   구매 취소금 + 쇼핑 지원금 = 총 지급액
   ```

3. **상품별 차이**
   ```
   지원금 비율은 상품마다 다름
   각 조별, 제품별 쇼핑 지원금에 따라 달라짐
   ```

## 📊 추첨 시나리오 예시

### 시나리오: 30명이 같은 상품 구매

#### 초기 상태
```
상품: 메가커피 30,000원 (지원금 1,500원)
신청자: 30명
```

#### 1차 추첨 (1~10번 회원)
```
조 구성: 1~10번 회원 (10명)
추첨 실행
  ↓
당첨: 3번, 7번 (각각 30,000원 구매 확정)
미선정: 1,2,4,5,6,8,9,10번
  
지원금 계산:
  당첨 지원금 합계: 1,500 × 2 = 3,000원
  미선정 총 구매금: 30,000 × 8 = 240,000원
  
  각 미선정자 지원금:
    (3,000 ÷ 240,000) × 30,000 = 375원
    10원 절삭 = 370원
  
  1번 회원: 370원
  2번 회원: 370원
  4번 회원: 370원
  ... (8명 모두 370원씩 지급)
```

#### 2차 추첨 (11~20번 회원)
```
조 구성: 11~20번 회원 (10명)
  ↓
당첨: 15번, 18번
미선정: 11,12,13,14,16,17,19,20번
  ↓
동일 방식으로 지원금 계산 및 지급
```

#### 3차 추첨 (21~30번 회원)
```
조 구성: 21~30번 회원 (10명)
  ↓
당첨: 23번, 29번
미선정: 21,22,24,25,26,27,28,30번
  ↓
동일 방식으로 지원금 계산 및 지급
```

#### 최종 결과
```
총 30명 중:
  - 당첨 6명 (3조 × 2명) → 실제 구매
  - 미선정 24명 (3조 × 8명) → 지원금 지급
  
매출:
  30,000원 × 6명 = 180,000원
  
지원금 지급:
  370원 × 24명 = 8,880원
```
