# 카테고리 · 상세페이지 표시 정리 (참고용)

**다음에 카테고리 목록이나 상세페이지가 안 나올 때 이 문서를 참고하세요.**

---

## 1. 전체 흐름 요약

| 구분 | 데이터 출처 | 표시 조건 |
|------|-------------|-----------|
| **네비/메뉴 카테고리** | Firestore `categories` 컬렉션 | 네비·사이드바용 (19개 등) |
| **상품 목록** | Firestore `products` 컬렉션 | URL에 `?category=카테고리ID` 있을 때만 해당 카테고리 상품 필터링 후 표시 |
| **상품 상세** | Firestore `products` 컬렉션 `.doc(상품ID)` | URL의 `?id=상품ID` 또는 sessionStorage `selectedProductId` |

## 2. 카테고리 목록이 안 나왔던 이유와 해결

### 2.1 원인

- **주소에 `?category=`가 안 붙음**  
  카테고리 링크가 상대 경로(`?category=ID`)였고, 서버 경로가 `/products-list`(`.html` 없음)라서 첫 클릭 시 브라우저가 주소를 제대로 만들지 못함.
- 그 결과 `products-list.js`의 `loadProducts()`에서 `urlCategory`가 `null`로 읽혀, 상품을 안 불러오고 빈 목록만 표시됨.

### 2.2 해결 (반드시 유지할 것)

1. **링크 주소 규칙**
   - **이미 products-list 페이지일 때**  
     → `?category=ID` 만 사용 (같은 경로에 쿼리만 붙임).
   - **다른 페이지(메인 등)에서 올 때**  
     → `/products-list?category=ID` 처럼 **전체 경로** 사용.

2. **적용 위치**
   - `js/nav-categories.js` → `getCategoryListHref(categoryId)`
   - `js/products-list.js` → `categoryListHref(categoryId)` (사이드바), 브레드크럼 링크

3. **클릭 시 반드시 URL로 이동**
   - `nav-categories.js`에서 `href`에 `category=` 가 들어 있는 `<a>` 클릭 시  
     `e.preventDefault()` 후 `window.location.href = categoryLink.href` 로 이동하도록 유지.

4. **products-list에서 상품 로드**
   - `loadProducts()` 는 **항상 `window.location.search`** 에서 `category` 를 읽음.
   - `?category=` 없으면 상품 목록은 비우고 "카테고리를 선택해 주세요"만 표시 (전체 상품 fallback 없음).

## 3. 상품 상세가 안 나왔던 이유와 해결

### 3.1 원인

- **product-detail.html에 Firebase 없음**  
  상세 페이지에서 Firestore로 상품 정보를 못 가져옴.
- **카테고리에서 상세로 가는 방식이 메인과 다름**  
  메인은 클릭 시 `sessionStorage` 에 상품 ID 저장 후 `product-detail.html?id=ID` 로 이동하는데, 카테고리(products-list)에는 그 로직이 없었음.

### 3.2 해결 (반드시 유지할 것)

1. **product-detail.html**
   - Firebase SDK 3개 + `firebase-config.js` 를 **상세 전용 스크립트보다 먼저** 로드 (순서 변경 금지).

2. **메인과 동일한 상세 진입 방식 (products-list)**
   - 상품 카드: `href="product-detail.html"` + `data-product-link="1"` + 부모에 `data-product-id="문서ID"`.
   - 그리드 클릭 위임:
     - `[data-product-link="1"]` 클릭 시  
       → `sessionStorage.setItem('selectedProductId', id)`  
       → `window.location.href = 'product-detail.html?id=' + encodeURIComponent(id)`  
   - 구현: `js/products-list.js` → `initProductGridClickForList()`, `createProductCard()`.

3. **상세 페이지에서 상품 로드**
   - `product-detail.js` → `getProductFromUrl()`  
     - URL의 `id` 또는 없으면 `sessionStorage.selectedProductId` 사용.
     - Firestore `products` 컬렉션에서 `doc(productId).get()` 로 **메인과 동일하게** 조회 후 표시.

## 4. Firebase "No Firebase App" / 앱 미초기화

### 4.1 원인

- `firebase.initializeApp()` 이 실행되기 전에 `firebase.firestore()` 를 호출함 (nav-categories, 상세 등).

### 4.2 해결 (유지할 것)

- **nav-categories.js**  
  `firebase.firestore()` 호출 전에  
  `firebase.apps && firebase.apps.length > 0` 인지 확인. 아니면 재시도 후 return.
- **product-detail.js**  
  `getProductFromUrl()` 등에서 Firestore 사용 전에  
  Firebase 앱이 준비됐는지 확인하고, 필요 시 짧은 대기 후 조회.
- **firebase-config.js**  
  `whenFirebaseReady` 에서도 `firebase.apps.length > 0` 확인 후 resolve.

## 5. 문제 발생 시 점검 순서

1. **카테고리 클릭해도 목록이 안 나옴**
   - 주소창에 `?category=숫자또는문자` 가 붙는지 확인.
   - 안 붙으면: `getCategoryListHref` / `categoryListHref` 가 위 규칙대로 링크를 만들고, 카테고리 링크 클릭 시 `location.href` 로 이동하는지 확인.
   - 붙는데도 빈 목록이면: 콘솔에서 `[products-list] loadProducts 시작 isCategoryPage=, urlCategory=` 로그 확인. Firestore `products` 컬렉션과 상품의 `category` 필드 값이 해당 카테고리 ID와 일치하는지 확인.

2. **카테고리에서 상품 클릭 시 상세가 안 나옴**
   - 상세 페이지 HTML에 Firebase SDK + `firebase-config.js` 가 있는지, 로드 순서가 맞는지 확인.
   - products-list 에서 `initProductGridClickForList()` 가 호출되는지, 카드에 `data-product-id` 가 들어가는지 확인.
   - 상세 페이지에서 `getProductFromUrl()` 이 URL 또는 sessionStorage 로 상품 ID를 읽은 뒤 `products.doc(id).get()` 하는지 확인.

3. **"No Firebase App" / 앱 미초기화**
   - `firebase-config.js` 가 해당 페이지에서 로드되는지 확인.
   - nav-categories / product-detail 에서 Firestore 사용 전에 `firebase.apps && firebase.apps.length > 0` 확인 로직이 있는지 확인.

## 6. 관련 파일 위치

| 파일 | 역할 |
|------|------|
| `js/nav-categories.js` | 카테고리 링크 생성(`getCategoryListHref`), 클릭 시 URL 이동, Firebase 앱 확인 후 categories 조회 |
| `js/products-list.js` | URL에서 category 읽기, loadProducts, 상품 카드 링크·클릭(`initProductGridClickForList`, `createProductCard`), `categoryListHref` |
| `js/firebase-config.js` | Firebase 초기화, `whenFirebaseReady` (apps 확인 포함) |
| `js/product-detail.js` | `getProductFromUrl()` (URL/sessionStorage → Firestore products.doc(id)), Firebase 앱 확인 |
| `product-detail.html` | Firebase SDK + firebase-config.js 먼저 로드 |
