# 솔라피(Solapi) 문자 서비스 연동 계획

> 내일 Firebase Blaze 연결 후 진행. Cursor로 단계별 구현 가능.

---

## 1. 구현 범위

| 기능 | 설명 |
|------|------|
| **회원가입 인증번호** | 전화번호 입력 후 인증번호 전송 → 사용자 입력 후 검증 |
| **회원가입 완료 문자** | 가입 성공 시 해당 번호로 "회원가입 완료" 안내 발송 |
| **마이페이지 알림 → 문자** | 주문 승인/배송/문의 답변/공지 등 알림 발생 시, 앱 알림과 동시에 문자로도 발송 |

---

## 2. 구현 순서

### 1단계: 문자 발송 API (Firebase Functions)
- **목표:** Solapi API Key/Secret은 서버에만 보관, 클라이언트 노출 금지.
- **내용:** Cloud Function 1개 작성 — 수신번호(`to`), 메시지(`text`) 받아서 Solapi로 발송.
- **환경:** Blaze 플랜 + 환경변수에 Solapi 키 저장.

### 2단계: 회원가입 인증번호
- 전화번호 입력 → "인증번호 전송" 클릭 시 Function 호출.
- Function에서 6자리 인증번호 생성 → Solapi 발송 → Firestore 등에 번호 저장(5분 만료).
- "인증 확인" 시 저장된 값과 비교 후 성공/실패 응답.
- `signup.js`의 `DEV_MODE` 해제 후 실제 인증 플로우 연결.

### 3단계: 회원가입 완료 문자
- 가입 최종 제출 성공 시점에, 가입한 전화번호로 "[10쇼핑게임] 회원가입이 완료되었습니다" 등 1통 발송.
- 1단계에서 만든 문자 발송 Function 재사용.

### 4단계: 마이페이지 알림 → 문자
- `createNotification()` 호출하는 곳(관리자/자동 처리)에서 알림 저장 후, 해당 회원 전화번호 + 알림 내용으로 문자 발송 API 호출.
- 또는 Firestore `notifications` 컬렉션 `onCreate` 트리거에서 새 알림 시 회원 전화번호 조회 후 Solapi 호출.

## 3. 예상 소요 시간

| 항목 | 시간 |
|------|------|
| 문자 발송 Function 1개 (Solapi 연동) | 30분~1시간 |
| 회원가입 인증번호 (발송+검증+프론트 연동) | 1~2시간 |
| 회원가입 완료 문자 | 15~30분 |
| 마이페이지 알림 → 문자 | 30분~1시간 |
| **합계** | **약 3~5시간** (설정·테스트 포함 시 반나절) |

## 4. 비용 참고

- **Firebase Functions:** Blaze 플랜 필수. 호출/실행량 적으면 월 몇 달러 수준.
- **솔라피:** SMS 1통 약 8~15원, LMS 약 40~50원. 건수에 따라 과금.
